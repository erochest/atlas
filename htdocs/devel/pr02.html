Overview

This section describes the various parts of the Atlas web site. It looks at how each part is organized and how it interacts with the other entities that comprise the site. First, there is the programming code that creates the site, whether the database or the static or dynamic pages. Second, it looks at the template system used to create the pages (again, both static and dynamic). Third, this looks specifically at how static pages are created and organized on the site. Finally, this looks at how dynamic pages are created and organized.
